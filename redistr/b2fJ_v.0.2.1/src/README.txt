Here there are sources for Java VM & jtools.

build.xml
  This ANT build file will build the jtools & class libraries that will be stored in lib folder.

java\classes
  Sources of class library (to replace standard Java classes)

java\tools
  Sources of jtools

javavm
platform\c64
  Java VM code (c64 folder contains platform specific files).

util
  Some code that is not directly related to JVM, but useful.
  e.g. dump.c prints out content of linked class files.
